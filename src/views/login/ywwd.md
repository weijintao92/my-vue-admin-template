2. 设置交换机管理地址
>对于不可傻瓜交换机（不可网管），是不能设置管理IP地址的！可网管的，你需要物理接触交换机，利用console线连接console口，用超级终端来进入交换机来设置管理IP地址！

```bash
System
interface 
vlanifip address 192.168.1.1 255.255.255.0
quit
```
